# X-TRACK_LQFP
@FASTSHIFT/X-TRACK


X-track的lqfp版本，将原先的AT32F40ACGU7更换为AT32F403ACGT7（即STM32F103C8T6那个最常见的48-pin LQFP 7X7封装） 重新布线过，第一版亲测正常工作，之后又优化了部分布线，减少焊接时可能出现的问题
